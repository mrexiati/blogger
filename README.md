### Installation

After cloning the repository, you can set up and run the application as follows:

1. **Instal dependencies:**
   ```sh
   yarn install
   ```

2. **Run the application on your local:**
   Generate production build
   ```sh
   yarn build
   ```
   Start the application on local host
   ```sh
   yarn start
   ```
